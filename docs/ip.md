### 新浪接口
获取当前城市信息
``` javascript
http://int.dpool.sina.com.cn/iplookup/iplookup.php?format=js
```
获取指定IP所在城市信息
``` javascript
http://int.dpool.sina.com.cn/iplookup/iplookup.php?format=js&ip=218.192.3.42
```
